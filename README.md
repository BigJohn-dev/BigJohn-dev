I build financial infrastructure for everyday Nigerians. My work cuts across **Go, Python, TypeScript, and Rust**: USSD wallets that bring crypto to feature phones, on-chain savings contracts, and payment SDKs. Trained through **Semicolon Africa's** year-long, build-first engineering residency, where shipping real products always beat coursework.

## 💼 Flagship Projects

| Project | What it does | Built with |
|---------|--------------|------------|
| 📱 **StellarUSSD** | Crypto wallet access from any feature phone. A Go backend translates USSD menu sessions into Stellar network transactions, with server-side key custody and sessions that survive dropped connections. Built for the millions of Nigerians excluded from smartphone-only wallets. | `Go` `Stellar SDK` `PostgreSQL` `Docker` |
| 🔄 **AjoChain** | Nigerian Ajo/Esusu savings groups, on-chain. Soroban smart contracts enforce payout schedules, rotation logic, and default handling at the contract level. No more relying on a group leader's goodwill. | `Rust` `Soroban` `Stellar` |
| 💱 **NairaRamp** | A typed NGN ⇄ USDC conversion SDK with a provider-agnostic adapter pattern and a clean quote → accept → settle flow. Rate-locking semantics enforced at the type level. | `TypeScript` `Node.js` `Stellar SDK` |
| 🏥 **MediPay Stellar** | A reconciliation layer for hospital and HMO settlements, tokenizing claim instruments on Stellar for deterministic processing. | `Rust` `Stellar` `TypeScript` |
| 💰 **Wallet System** | Digital payment backend handling accounts, transactions, and budgets. Money stored as integer minor units, because floats and money don't mix. | `Python` `PostgreSQL` `REST` |
